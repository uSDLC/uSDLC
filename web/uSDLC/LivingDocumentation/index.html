Page Comments

A visitor may not want to or be authorised to update a page. They can leave comments that are attached to the page as side-notes. Every comment raises a task until it is flagged as processed.

History

For developers we recommend keeping uSDLC within the application project. Use your version control system to keep the documentation and code synchronised and safe.

uSDLC is designed to be used by all stake-holders in the software development lifecycle. So it provides support for saving, reviewing and reverting changes. Every change made to the document is recorded against the user who made it. Changes can be compared visually and sections reverted to an earlier version.
