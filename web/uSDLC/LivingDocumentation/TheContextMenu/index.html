The Context Menu

An Activity Shortcut System

Almost every section of the screen has a context menu - that changes for each type.

These menus can be activated by right-clicking the mouse or pressing and releasing the Alt key while a section is in focus.

Once the context menu has been displayed, navigation can be achieved with mouse or the arrow keys. Most context menu items have a keyboard shortcut that can be used. The shortcut mnemonic is displayed with the menu item.
