Contents Tree

A visual aid

Within the content tree the arrows keys will move up and down as well as in and out of child pages.

Content Access

Clicking on a node or pressing Enter while a node is in focus will replace the contents in the main window. Using Alt-click or Alt-Enter will open the node in a new pop-up window. Once a window has been opened for a link it's size and position is saved and all future access will go to this window. To open the content again in the main page, use Ctrl-click or Ctrl-Enter. All these features are also available front the context menu.
