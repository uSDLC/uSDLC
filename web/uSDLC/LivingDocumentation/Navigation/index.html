Navigation

The art of knowing where to go

As a document gets larger it becomes more difficult to find the right place. uSDLC provides navigation by Content tree, search and reports.

In using a mouse, navigation is self-evident. Click on a heading or a section and it will highlight. The highlight will be different if the page is in Run or Edit mode. Clicking on the logo will return to the front of the document. The content tree index can be opened and closed and pages selected directly. Selecting the scroll bar button or clicking within the scroll bar region will enable scrolling functionality. Clicking on any section group will enable keyboard navigation for that group - just as Tab does.

Keyboard navigation is just as simple. Use tab to move beween the contents, index and scroll sections. The action of the arrows key varies for each section.

Use up and down in the content section to highlight the adjacent section. If the section is off the screen is is scrolled into view.

Bookmarks

The title for the page is also a direct link to the page. Clicking on it will open the page in a new window. Save it as a bookmark or copy the address for reference elsewhere. Alternatively use the context menu to copy the url to the system clipboard.

Mind Mapping

uSDLC has an integrated version of Freemind - a Java based mind-mapping solution. It can be used to both visualise and create the document structure in a graphic fashion that suites many creative thinkers.
