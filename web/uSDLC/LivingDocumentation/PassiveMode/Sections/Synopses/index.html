Synopses

Building a Page from Bits

Any section with only one link at the top and that to a child page will be converted to a synopsis. When looking at a synopsis you can see the title, sub-title and introductory sections from the child page.

Selecting a Synopsis highlight and change the contents to the current page (rather than a synopsis of the child). The link will then be live so that you can follow it to the child.

A synopsis section will behave as per normal. You can edit, move, combine, split, cut, copy and paste. A cut/copy will save the contents of the child pages as well as the reference section.

If text is added before the link or additional links placed in the section, it will cease to be a synopsis and revert to a normal section.
