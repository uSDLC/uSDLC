Section Classes

Yes, but what do I really mean?

Every section has a class - some implicit based on location and content, others explicit. There are classes for synopses, notes, to-do lists, assignments and data.

Synopsis Class

The first section on a page is an implicit Synopsis class. It should not have internal links and will display on the parent page as a summary of this book section. It has a cream background by default.

Margin Note Class

These sections will not become part of the book if one is created. In many reproductions (such as PDF) they will become part of the note system. By default the background is a burnt orange.

Alert Class

Sections with this class are not part of the finished document. They are there as place-markers for tasks that are not yet fleshed out.

Implement Class

Classes that are ready for immediate implementation.

Suspended Class

Section that is not to be implemented, tested or otherwise worked on.
