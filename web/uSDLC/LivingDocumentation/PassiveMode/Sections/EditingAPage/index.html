Editing a Page

Mastering the Word Jigsaw

Almost all page editing is about creating, editing or moving sections.

Inserting a New Section

The context menu can be used to insert a new blank section relative to the section in focus. Ctrl-I is the shortcut to insert a new section above, while Ctrl-A will append the new section below the focus.

A New Page

When creating a new page a single empty section will come for free. Clicking on this or any section will give it focus and highlight colouring.

The Title Section

The title section can be edited in the same fashion as content sections. It does have a more limited context menu.

Uploading Images

The easiest way to add images from a brower based page is to use copy and paste. CKEdit puts a reference to the image into the HTML. This is not suitable if the original page is not available. It is often best to carry the image with the document. The image dialog includes an upload button. Click this and browse for the image you want to display.

Moving Sections

Sections can be moved around the page by either dragging them with the mouse or by holding the <Ctrl> key down and using the up or down arrow keys to push the highlight section up or down the stack. In all cases the focus stays with the section being moved. To abort once a drag has started, use the <Esc> key.

Convert Section to Page

A section can be turned into a new page from the context menu or with Ctrl-T. The first paragraph or heading becomes the title, while the rest of the contents becomes the first paragraph. The new page will use the same name as the title - with spaces removed. A reference to the new page is kept in the parent. The index is updated.

Merging Sections

Sections can be merged by holding down the <Alt> key along with the up and down arrows to merge the focus with the section above or below. The merged section gains the highlight.

Creating Sections from A List

Sometimes it is useful to enter a list of items and break them up into sections later. The breaking up can be achieved with a context menu command or the Ctrl-S key. If the content is made up of a numbered or bullet list, each item is given it's own section. The final page structure will reflect a multi-level list. Otherwise each paragraph becomes a section. This is a useful technique for entering information from emails or other external sources.

Page Locking

When a user chooses to edit a section it is locked. Others may edit other sections. They will be given a message if they attempt to edit a locked section. If a page has changed in the time between a user has loaded it and they choose to edit, insert, move or delete a section then it will be reloaded prior to the operation. If the section being manipulated has been deleted, an error message is displayed.

A section lock is held until the user saves the section being edited or loads another resource. If a lock is over 1 hour in age, anyone who gets a waiting for lock dialog can also choose to over-ride the lock and edit anyway. The authorisation system has an optimistic group. Anyone in this group is allowed to over-ride a lock at any time.

Internally page changes are queued and applied in order of submission.

Attempting to save a section on a page where the lock has been over-ridden will display an error dialog. The section that could not be updated is placed in the clipboard for re-use.
