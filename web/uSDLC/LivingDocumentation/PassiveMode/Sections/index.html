Sections

The Hub

A page is made up of discrete sections. These sections can be edited, copied, cut or run. If a section only includes a link to a page or script, the target is shown in-line.

At any one time only one section can be in focus. It has a different background and/or text colour to make this obvious. The section in focus is saved so that when you return to the page it is restore to view and focus.

Synopses

Minimalism

A single section or the complete page can be displayed in minimal mode. Only the first line/heading is visible. Clicking on a minimal section will expand it,as will the right arrow once it is in focus. The left arrow will minimise an expanded section. Alternatively, use the context menu. If a section is minimised, any following sections with a lower header level will also minimise - and indent to show the relationship. In other words, clicking on a H2 header will also minimise and H3 to H6 headers between it and the next H2 header.
