The Clipboard

Movin' Stuff Around

The clipboard with the ubitiquitous cut/copy/paste is ingrained in modern computing. When editing a section or source code you can highlight and use the system clipboard as usual. When in page display mode you can cut, copy or paste the section highlighted.

 As usual you can use key-strokes or alternatively the context menu. By default the standard ^C copies, ^X cuts and ^V pastes the most recently added item. You can change this from the configuration page. Right-clicking on a section or pressing the context menu key will bring up a menu that includes the clipboard.

There is no delete. Anything you cut to the clipboard will remain there unless it is pasted elsewhere. From the context menu you can get to the last 5 items to be pasted. To get to more, go from there to the extended clipboard page.
