Site Updates

When the site structure needs to change

uSDLC records all pages as standard HTML files in directories matching the tree navigator. The result is documentation that can be walked without the software. The down side is that sometimes page structure is out-dated. This page contains a core script that walks a uSDLC site and makes changes on request.

Parser Sample

The basics are simple. Create methods called page and section that will be called as the parser walks the uSDLC document tree (including projects). Lastly, call run.uSdlcPageParser to trigger the walk-through.

Clean HTML

The parser above is for one-off changes. In most cases it is better to enter into a permanent process that will correct recurring problems.
