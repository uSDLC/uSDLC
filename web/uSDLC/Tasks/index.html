Tasks

What to do, what to do...

Every page, section or actor link is a task. Tasks are used to track and assign work.

Task Report Link Creation

uSDLC comes equipped with a Task Report Actor. You generate a report from the Reports tab on the create link box - available from the link button on the tool-bar. The first field is a list of reports defined in /rt/reports. Selecting a report will fill in the link URL.

Task Report Location

While is it possible to generate report pages anywhere, convention is to put them in the home pages chapter. When a member is added to the system they are given a home page from a configuration template. In the standard configuration this template includes a link to reports relevant to them member - tasks assigned to them, tasks completed, etc.

A project manager or team leader may add reports that represent their interest areas. The members area also includes a member called Project. Here resides reports of general interest.

Clicking on a report link will generate the report and present it for review.
