Unifying the SDLC

with Document Driven Development (D3)

Modern software development methodologies, techniques and technologies are now ready to support the unification of the software development life-cycle. This system works step-by-step towards that goal. While the term SDLC is as old as the IT industry, it is still a valid term for encompassing all the processes that are required to create and maintain a software system. This system uses agile methodologies to build upon the Lean principals to reduce waste and increase productivity.
