Java Actor

Classes as Scripts

Java is not a scripting language, but since uSDLC runs on the JVM, it is easily integrated. uSDLC will launch Java classes that inherit from usdlc.actors.JavaActor or JUnit tests.

Classes inherited from JavaActor have env and doc instance variables set and call the run method. Use the convenience methods:

  • alert(messageFile) Display an alert message.
  • end() close an outstanding tag()
  • escape("a < 12") writes text to the browser. HTML characters are encoded so they will display instead of processed.
  • highlight("blue") Highlight a section with a coloured string on the left
  • requestBody() returns post data from the browser.
  • script("window.location.href=window.location.href") - run a script from inside a browser
  • tag("div class='myClass'") is used to open any tag. You can optionally add attributes
  • text("<div>inside a div</div>") writes text to the browser. HTML tags will be processed

If you have legacy junit tests, use the Groovy ant-builder to instrument them.

new AntBuilder().junit {
    test(name : 'mypathtojunittestfiles')
}
