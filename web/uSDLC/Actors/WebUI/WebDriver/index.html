Selenium WebDriver

The New Age Browser Driver

Selenium WebDriver will allow an application to manipulate a browser without the need if an intervening proxy server.

First we have to connect to a browser. Use the full class name of the driver or any shortcut defined in Options /  Configuration - the standard being chrome, firefox, ie or htmlunit. If none are set then the first time a command is issues the default browser loads - also as defined in the configuration.

Next we open a page and click on a link to open the Sandbox.
