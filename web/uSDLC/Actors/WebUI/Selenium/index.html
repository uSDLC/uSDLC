Selenium

Real browsers real quick

Selenium is a system to test the UI on real browsers. It has drivers for Chrome, Firefox, Internet Explorer, iPhone, Android and HtmlUnit. Scripts can be auto-generated in Firefox with either the Selenium IDE or Sauce Builder. Selenium supports grids for parellel testing and Sauce Labs extends that to allow testing on the cloud.

Without further ado - on to skinning the cat. The  Selenium IDE or Sauce Builder can record and save a script in a language of your choice. The easiest is to use the Groovy DSL that comes with uSDLC. The first script will load the uSDLC site and come to this page and highlight the first section.

Any Selenium command can be sent. The server starts when the first test is triggered.

As a third approach, have a look at Geb. It uses the Selenium web drivers, so can operate with Firefox, IE, Chrome or HtmlUnit.
