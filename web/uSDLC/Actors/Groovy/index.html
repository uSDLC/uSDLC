Groovy Actor

Without which uSDLC would not function

uSDLC is written in Groovy, so this actor is the most integrated. Many other JVM tools - such as junit and canoo rely on a groovy script as a proxy.

A groovy actor is a groovy script with the following global variables available.

  • env - the actor environment as documented on the parent page.
  • doc - an extended markup builder for easily generating xml or html
  • gse - the scripting engine used to create this instance
  • include - a helper to include one script inside another
  • template - a helper to include a template from the template directory inside a script
