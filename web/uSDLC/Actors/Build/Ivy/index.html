Ivy

Everybody depends on somebody

uSDLC supplies dependency management through the services of Ivy and Maven. For the non-JVM world, directly downloads of dependencies are also supported.

Dependencies can be manually updated from the options page of the same name. For non-core dependencies, the system will check to make sure a copy is available before an actor that needs then is run.

Dependency actors have an extension of ivy. Any generic or ivy specific ant task can be used as a command. See the options page for examples. There are a number of convenience functions that will fetch dependencies for required, optional and source targets. The source target also downloads the documentation if it is available.

In addition to retrieving dependency trees from Maven repositories, the class can download a file directly over http.

All dependency retrievals write the results to a persistent log file in case a later review is required.
