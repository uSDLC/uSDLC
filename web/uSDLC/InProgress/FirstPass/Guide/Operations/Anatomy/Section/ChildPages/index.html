Child Pages

Every page you see here is a child page, although the root child pages are children of a project. Parent pages show a synopsis of the child page using the contents of the first section.

Creation

A section can create a reference to a child page in one of three ways:

  1. Select New | Page from the menu (or type NP)
  2. Select Section | Insert | Above or Below from the menu (type SIA, SIB or Ctrl-I, Crtl-A)
  3. Add a link using the editor on an existing section.
