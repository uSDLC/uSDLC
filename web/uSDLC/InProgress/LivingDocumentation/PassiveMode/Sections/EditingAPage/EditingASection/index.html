Editing a Section

Changing the way we feel

To edit the contents of a section, double-click on it or right-click for the context menu and select Edit Section or press control-E. The current contents of the section are loaded into a powerful wysiwig editor called CKEdit. The page will scroll up so that the editor window is at the top. The editor will expand to fit the content size. A section can be of any size, but convention and convenience limit the normal section to one paragraph or equivalent. Once you leave the editor the focus is placed on this new section.

Secton Id

When a section is created it is assigned an Id unique for the page.

Spell Checking

CKedit uses an on-line checking facility from spellchecker.net. It will only work if the computer running the uSDLC client has Internet access.

Macros

Any documentation will have commonly repeated text - be it the software name, the client or other common repetitive elements. uSDLC has augmented the editor with a macro facility.
