Page Hightlights

Where is everything?

Some of the page features are in-your-face while others are subtle.

Theme

uSDLC uses the jQueryUI Theme Roller. By selecting a new theme from the configuration page you can change to look and feel of the page.

Page Title

At the top of the page is a title section. It can be edited as any other section. The title, sub-title and first paragraph are used in the linked synopsis on parent pages. The main title is linked to the current page. It can be used to create a URL that goes directly to this page.

Project Logo

To the right of the title is the uSDLC logo, or whatever your system replaces it with. Clicking on the logo will return you to the jacket of the document.

Button Bar

To the left of and just below the title is a button bar containing important system-wide actions.

Below the button bar is the index. This can be used to get an overview of the document and as the primary navigation feature.

The bulk of the page is content. Content is divided up into sections that can be edited individually. A section is typically a paragraph, but can also be a table, list, image, a form or any combination of these.

The content window has a scroll-bar. This scroller will float above any modal lock so that you can view any part of the content at any time. The window itself will not have either horizontal or vertical scroll bars. If needed, the index will also display a scroll-bar.

The first section in the content is also used in the synopsis on the parent page. It should contain a short overview and no links.

Sections

Sections can (and must) include links. Links will all present differently. Links to external sites will look like standard web links. Links for child pages do not have the underscore and are in a darker blue unless hovered over. Links to actors are brown unless they are active - when they take on background colour changes to show state.

Modal Background

Sometimes it is necessary to block page access while another operation is in progress. The most common is editing. The rest of the page darkens and it is not possible to use the button bar, index, sections or scroll bar. If you resize the window the modal overlay resizes accordingly.

Footer

The page footer display information relevant to the page - as in the last editor, assignee(s), last date changed, last date run, run state (passed or failed).
