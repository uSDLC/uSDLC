uSDLC Stand-Alone

No install - just run

The stand-alone version is designed for systems that do not have an Internet or repository connection. It is a single uSDLC-full.jar file. Copy it into an empty directory in your project and run it with

java -jar uSDLC-full.jar

It will unpack itself - by only over-writing files that are newer in the archive than on disk. This way you can update uSDLC on an existing project safely.

Snapshot

It is possible to take a snapshot of the uSDLC tree at any time and create a self-extracting jar file by running this section. It can be copied to any machine with Java installed and run to create a copy of the current system. The same jar can be copied again at any time. When it is run it will update the uSDLC tree by extracting only the newer files.
