Content

A section that is not in the standard format of child page or instrumentation reference can be created and formatted to suite.

Creation

A plain section can be created in one of two ways:

  1. Select New | Section from the menu (or type NS)
  2. Select Section | Insert | Above or Below from the menu (type SIA, SIB or Ctrl-I, Crtl-A) and select the third tab, called Section.
