Maintenance

Page Editing

Pages are edited by section. Click on a section and it will highlight. Either select edit from the context menu or if the page is in edit mode you can double-click. The HTML editor is a complete word processor. Use any of the functions to brighten up your page. Click the save icon on the tool-bar to send the changes back. If all is well the section will receive the green strip of success.

Creating New Pages

Click on the tool-bar link item in the editor. Type in a readable name while leaving the protocol as local. When you save the section the link will be converted to a readable URL. Clicking on the link will move to the new page empty but for the template.

Inserting New Sections

A new page will always have an empty section. From here you can insert a new section above or append one below. Right-click on a section. It will turn yellow and display a context menu. Two of the options are insert and append.
