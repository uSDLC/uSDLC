The Vision

Where it all begins

All software projects are started by a need as seen by the client. A green-field project is driven by a need to improve a process or to provide a new service (such as a website). Existing systems get outdated by client requirements changes, hardware updates or other factors. Most software systems are inflexible and a re-implementation is needed. One of the core aims of this project is to extend end-of-life to it's natural limit.
