uSDLC Networking

Creating a community

uSDLC is designed as a distributed system. It has multiple methods defined here to seed a new system and make it part of the network.

Single File Export

It is possible to take a snapshot of the uSDLC tree at any time and create a self-extracting jar file by running this section. It can be copied to any machine with Java installed and run to create a copy of the current system. The same jar can be copied again at any time. When it is run it will update the uSDLC tree by extracting only the newer files.
