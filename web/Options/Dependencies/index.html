Dependencies

Dependent children that work - an oxymoron?

uSLDC will work in it's base state if all actors are written in Groovy or Java. To environments where other computer languages are king, we need to download the optional dependencies. Most jvm dependencies come from maven repositories. For other parts of the world, downloads and installation  can be either automatic or prompted.

Setup

Required

uSDLC comes with the core Groovy dependency installed. Many actors require optional components. You can also use this link to update to the latest version of Groovy. If you work within an IDE you may also choose to download the source and javadoc. Ivy is also required to load from Maven. Many packages use Apache Commons logging. Required

Web

Selenium runs front-end tests from a real browser (many brands). HtmlUnit tests web pages in a headerless fashion (without a browser). Hue Doj provides jquery-ike access to the HtmlUnit DOM. This is a necessary module if your project has a web/browser based user interface. Web

Optional

A lot of optional components are only required under specific conditions.

Appengine

Only required if instance is to run on Appengine.

Immediate

Add items temporarily to this actor for immediate collection.
