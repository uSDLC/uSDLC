Project Template

When you add uSDLC to an existing project or create a new one, the basic structure is taken from the default template here.

The Guide page has By example, How-to and Reference children. Most of the content is from references from guide sections in the Design document.

Design is a page of type Project. For most projects you fill it with vision points - one per section.

Every project needs an Environment. Pages deleted from the navigation tree are moved to Trash. All users for whom this project is home have pages here also.
