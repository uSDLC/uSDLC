Guide

Templates

Every page, section definition or element in uSDLC can be modified. There are templates for message boxes, forms, and new pages or projects.

Guide

Dialog Boxes

Dialog box templates are read from a subject specific page. The name is matched from the first <H1> element in a section. This is also used as the title of the dialog unless it is followed by a second <H1> element. Names with a colon can be used to create a multi-tabbed dialog box. A single page can be displayed by specifying the full name. A tabbed list needs only the text before and including the colon. "Section:" will display a tabbed list of section forms, while "Section: General" will only display one.
