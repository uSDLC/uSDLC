User Administration

Users can log in, log out and change password from the main menu. In addition an administrator can create new users.

Each user gets a login page and a home page. The login page sets user details and group memberships in the session. The user name above the left navigation tree is also a link to a user home page.

Users can be created in any project, but their access to all project is by group membership, not parent project.
