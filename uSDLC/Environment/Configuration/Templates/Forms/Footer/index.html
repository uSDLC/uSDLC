Footer

A standard page footer will have a link to the parent page and a more... command. The latter allows you to add Setup, Cleanup, Instrument, Groups or general Script.

If you add scenarios to a page, Setup and Instrument scripts are automatically created.

Setup, Cleanup, Instrument and Script entries can be in any supported script language although Coffeescript is used for Setup and Instrument on pages used for scenarios.

Groups is a csv file containing a list of groups that are allowed to access this and child pages.
