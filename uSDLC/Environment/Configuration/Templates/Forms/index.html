Forms

Forms for all ages reside here, broken into groups. A form is created as a section, but it is displayed in a modal dialog box when needed.

Most forms are populated by and trigger client side scripts. If an input element has a activate attribute, it points to a function in the usdlc namespace. For a text element, jqueryUI auto-complete is useful.
