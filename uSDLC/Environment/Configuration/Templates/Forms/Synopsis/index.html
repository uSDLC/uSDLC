Guide

Synopsis

The first section on any page is a synopsis that is displayed in the referencing parent page. uSDLC facilitates organised design by offering a form of ubiqitous language fields that will fill the synopsis.

This form is only an option. You can always press escape and edit the synopsis section yourself.

The As a // I want // so that ubiqitous language introduced in behaviour driven development (BDD) provides the default questions for Project through to Feature pages. This can be changed to anything your team requires by editing any or all of the forms. Input elements have an alt attribute that is used for the bold ubiqitous sentence start.
