uSDLC Stand-Alone

No install - just run

As a uSDLC Developer
I want be able to create a stand-alone single jar installation solution
so that I can easily create a new system without additional Internet downloads

The stand-alone version is designed for systems that do not have an Internet or repository connection.

  • Create a directory uSDLC at the same level as your projects.
  • Download:
    • uSDLC-full.jar for a running system and;
    • uSDLC-optional-jars.zip for a sandbox (no internet) system.
  • From a prompt, run:
    • java -jar uSDLC-full.jar

It will unpack both archives - by only over-writing files that are newer in the archive than on disk. This way you can update uSDLC on an existing project safely.

If you don't download the optional jar archive, uSDLC will fetch the jars on demand.

Snapshot

It is possible to take a snapshot of the uSDLC tree at any time and create a self-extracting jar file by running this section. It can be copied to any machine with Java installed and run to create a copy of the current system. The same jar can be copied again at any time. When it is run it will update the uSDLC tree by extracting only the newer files.
