Changing Page Structure Support

The forms used for both sections and the synopsis on a new page are available for change at Environment / Configuration / Templates / Forms. Each section is a form that can be maintained directly from within uSDLC.

Form Title

Any form can be displayed by page and name. For tabbed forms the name is patterned on "Form: Tab". Look at the New Section page for an example. When adding a new section directly, tabs are displayed for each page type. The types are bypassed if the user uses the New option from the menu and chooses the section type directly.

Editing a Form

The built-in editor provides form and field selections. For synopsis forms, you will need to edit the source and add alt attributes. These are displayed in bold as part of the synopsis.
