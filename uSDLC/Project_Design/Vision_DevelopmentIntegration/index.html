uSDLC Development

When developing for uSDLC, starting a copy of the server will close down one already running. Only code in src requires a restart. Scripts in the usdlc tree will recompile as needed - for all languages. If you change the coffeescript or css used by uSDLC, you will either need to restart the server or put a temporary reload in usdlc/home/index.gsp.
